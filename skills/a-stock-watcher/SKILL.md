## 📜 投资宪法（INVESTMENT CONSTITUTION）

**这是铁律，不可违背，不可例外，违者必败**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
                           【 投 资 宪 法 】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  第一条【止损铁律】
  ├─ 任何单笔交易，亏损不得超过总资金的 2%
  ├─ 触发止损，必须立即执行，不得犹豫
  ├─ 不得以任何理由（"基本面没变"、"还会涨回来"）持有亏损仓位
  └─ 违反此条者，市场会惩罚你，惩罚方式是清零

  第二条【仓位上限】
  ├─ 单只标的仓位不得超过总资金的 20%
  ├─ 同一板块持仓不得超过总资金的 30%
  ├─ 弱势市场中，仓位不得超过 50%
  └─ 绝对禁止：全仓、重仓单一标的

  第三条【买入时机】
  ├─ 只在以下情况买入：
  │    ① 上升趋势确认（价格站稳20日均线上方）
  │    ② 回踩关键支撑位企稳
  │    ③ 突破关键压力位放量
  │    ④ 成交量放大至近5日均量的1.5倍以上
  ├─ 绝对禁止：追高买入（当日涨幅超过5%不买）
  └─ 绝对禁止：在下降趋势中抄底

  第四条【卖出时机】
  ├─ 止盈：上涨15-20%减仓1/3，30%再减1/3
  ├─ 移动止损：利润回撤50%时，触发止盈
  ├─ 止损：亏损2%执行止损
  └─ 强制卖出：连续3天下跌，不管盈亏

  第五条【禁止行为】
  ├─ 禁止：听消息炒股
  ├─ 禁止：全仓进出
  ├─ 禁止：逆势加仓摊平成本
  ├─ 禁止：频繁交易（每周交易不超过3次）
  └─ 禁止：亏损时加仓，盈利时不敢加仓

  第六条【仓位管理】
  ├─ 100万资金，建议持有3-5只标的
  ├─ 每只标的初始仓位10-15%，最多加仓至20%
  ├─ 保留30%现金作为子弹
  └─ 熊市/震荡市：仓位30-50%

  第七条【选股标准】
  ├─ 只买：行业龙头/细分市场前三
  ├─ 只买：有业绩支撑（营收/利润正增长）
  ├─ 只买：流动性好（日成交额>1亿）
  └─ 回避：ST股、亏损股、流动性差的标的

  第八条【每日必检】
  ├─ 检查持仓标的是否触发止损
  ├─ 检查大盘趋势是否改变
  ├─ 检查持仓标的是否出现卖出信号
  └─ 记录当日操作和理由

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 止损矩阵

```
单笔交易风险控制：
├─ 总资金 2% = 20,000 元（单笔最大亏损）
├─ 止损比例 5-8% = 对应止损价位
└─ 例：买入价10元，止损8%，止损价9.2元

持仓风险控制：
├─ 单只标的最大亏损 = 总资金2% = 20,000元
├─ 全部持仓最大亏损（同时触发）= 总资金10% = 100,000元
└─ 超过此值，必须减仓
```

## 🎯 精确进出场标准

### 买入标准（必须全部满足）

```
买入条件检查清单：
□ 1. 趋势确认：价格 > 20日均线
□ 2. 板块强势：所属板块指数 > 20日均线
□ 3. 缩量回踩：回调时成交量萎缩至均量的50%以下
□ 4. 企稳信号：回踩支撑位后出现阳线
□ 5. 放量确认：当日成交量 > 近5日均量1.5倍
□ 6. 基本面：无重大利空，业绩稳定

买入价格：
├─ 激进买入：回踩支撑位时买入5成仓
├─ 确认买入：突破关键位放量时买入3成仓
└─ 回踩确认：突破后回踩不破时买入2成仓
```

### 止损标准

```
硬性止损（必须立即执行）：
├─ 价格止损：买入后下跌 8% 执行止损
├─ 时间止损：买入后 5 个交易日不涨，执行止损
└─ 趋势止损：价格跌破 20日均线，执行止损

软性止损（考虑执行）：
├─ 移动止损：利润回撤 50% 启动保护
└─ 板块止损：所属板块指数跌破 20日均线，考虑止损
```

### 止盈标准

```
阶梯止盈法：
├─ 第一目标（+15%）：减仓 1/3，剩余仓位移动止损至成本价
├─ 第二目标（+25%）：再减仓 1/3，剩余仓位移动止损至+15%
└─ 第三目标（+35%+）：持有不动，跌破 20日线全部清仓

移动止损：
├─ 利润 10% 以内：止损线 = 成本价
├─ 利润 10-20%：止损线 = 成本价 + 利润的 50%
├─ 利润 20%+：止损线 = 成本价 + 利润的 70%
└─ 绝对不允许：盈利单变成亏损单
```
