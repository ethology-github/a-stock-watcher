# A股守望者 (A-Stock Watcher)

**一个融合利弗莫尔择时智慧与巴菲特价值理念的A股智能投研系统**

---

## 🎯 项目简介

A股守望者是一个多Agent协作的智能投研系统，由4个专业Agent组成：

| Agent | 角色 | 职责 |
|-------|------|------|
| **情报员** | 新闻雷达 | 盯财经热点，刷新闻热搜，识别重大事件 |
| **研读师** | 研报助手 | 批量读券商研报，提取目标价和核心逻辑 |
| **精算师** | 数据分析师 | 接入实时行情，技术面和基本面解读 |
| **守望者** | 投资大师 | 统筹决策，给出大师级市场分析和操作建议 |

### 核心特色

```
┌─────────────────────────────────────────────────────────────────┐
│  守望者 = 40年实战投资大师                                        │
│                                                                 │
│  融合两大投资流派：                                               │
│  ├─ 利弗莫尔：趋势择时、关键点买入、止损纪律                      │
│  └─ 巴菲特：价值投资、合理估值、行业龙头                          │
│                                                                 │
│  特点：                                                          │
│  ├─ 不是信息搬运工，是有独立判断力的大师                          │
│  ├─ 每一次操作都有：价格、止损、目标、仓位                        │
│  ├─ 精确的入场出场时机判断                                        │
│  └─ 100万资金，目标年化30%+                                       │
└─────────────────────────────────────────────────────────────────┘
```

## 🔧 快速开始

### 1. 安装依赖

```bash
pip install mootdx akshare pandas
```

### 2. 配置环境

```bash
# 设置数据源（可选）
export MOOTDX_SERVER='124.70.199.56:7709'
```

### 3. 运行数据采集

```bash
python scripts/a_stock_data.py
```
