# 投资宪法速查卡

**100万资金 · 铁律不可破**

---

## 仓位规则

| 规则 | 上限 | 说明 |
|------|------|------|
| 单只标的 | 20%（20万） | 初始10-15%，可加仓至20% |
| 同一板块 | 30%（30万） | 避免集中风险 |
| 保留现金 | ≥30%（30万） | 子弹，永远要有 |
| 持仓数量 | 3-5只 | 分散但不分散 |

---

## 止损规则

| 类型 | 规则 | 执行 |
|------|------|------|
| 单笔止损 | ≤2%（2万） | 触发即执行 |
| 价格止损 | -8% | 触发即执行 |
| 趋势止损 | 跌破20日线 | 触发即执行 |
| 时间止损 | 5日不涨 | 考虑执行 |

---

## 买入规则

```
✅ 可以买入：
□ 上升趋势（价格 > 20日均线）
□ 回踩支撑位缩量企稳
□ 突破压力位放量
□ 成交量 > 均量1.5倍

❌ 禁止买入：
□ 下降趋势中抄底
□ 当日涨幅 > 5% 追高
□ 无业绩支撑的标的
□ ST股、亏损股
```

---

## 止盈规则

| 涨幅 | 操作 | 止损移动 |
|------|------|----------|
| +15% | 减仓1/3 | 止损移到成本价 |
| +25% | 再减仓1/3 | 止损移到+15% |
| +35%+ | 全部/持有 | 止损20日线 |

---

## 禁止行为

```
🚫 听消息炒股
🚫 全仓进出
🚫 逆势加仓
🚫 亏损死扛
🚫 频繁交易（每周>3次）
🚫 盈利单变亏损单
```

---

## 操作检查清单

```
买入前必检：
□ 趋势确认？
□ 止损位设定？（精确价格）
□ 单笔亏损 < 2万？
□ 仓位 < 20%？
□ 风险收益比 ≥ 1:3？
□ 持仓周期明确？

持有中必检（每日）：
□ 触发止损？
□ 趋势改变？
□ 达到止盈位？
□ 板块走弱？
```
