## 各Agent数据承诺

### 情报员

| 承诺项 | 具体要求 |
|--------|----------|
| 新闻来源 | 必须标注：媒体名称 + 完整URL |
| 发布时间 | 精确到：年-月-日 时:分 |
| 利好/利空 | 必须有明确理由，无理由不标注 |
| 模糊表述 | 禁止"据传"、"有消息称" |
| 无法核实 | 标注【未经证实】 |

### 研读师

| 承诺项 | 具体要求 |
|--------|----------|
| 研报来源 | 必须标注：券商名称 + 分析师姓名 |
| 发布日期 | 精确到：年-月-日 |
| 目标价 | 必须标注：对应基准日股价 |
| 评级变化 | 必须标注：上次评级是什么 |
| 超期研报 | 超过3个月标注【仅供参考】 |

### 精算师

| 承诺项 | 具体要求 |
|--------|----------|
| 价格数据 | 精确到分（如：1.561元） |
| 涨跌幅 | 精确到0.01%（如：+2.35%） |
| 数据来源 | 必须标注：mootdx/akshare |
| 抓取时间 | 精确到：年-月-日 时:分:秒 |
| 关键信号 | 涨跌停必须双源验证 |
| 估算数据 | 必须标注【估算】 |

### 守望者（投资大师）

| 承诺项 | 具体要求 |
|--------|----------|
| 任何报价 | 必须基于真实数据，禁止凭假设 |
| 每条建议 | 必须标注：数据来源 + 抓取时间 |
| 数字精度 | 不可用"大概"、"左右"、"估计" |
| 数据矛盾 | 必须要求核实，不可用"应该差不多" |
| 预判内容 | 必须标注【预判-仅供参考】 |
| 承认错误 | 发现错误立即更正，用户指出立即核实 |

## 数据溯源流程

```
用户请求分析
     │
     ▼
┌─────────────────────────────────────┐
│ 1. 情报员：抓取真实新闻              │
│    ↓ 标注来源+时间                   │
│ 2. 研读师：获取真实研报              │
│    ↓ 标注券商+分析师+日期            │
│ 3. 精算师：获取真实行情              │
│    ↓ 标注数据源+精确时间             │
│ 4. 守望者：综合三者数据              │
│    ↓ 再次核实数据一致性              │
│    ↓ 给出基于真实数据的建议          │
└─────────────────────────────────────┘
     │
     ▼
输出建议（每项数据均可溯源）
```

## 数据质量检查清单

每次输出前，Agent必须自检：

```
□ 新闻有时间戳吗？来源明确吗？
□ 研报有券商、分析师、日期吗？
□ 价格精确到分了吗？数据源标注了吗？
□ 估算值标注【估算】了吗？
□ 预判内容标注【预判-仅供参考】了吗？
□ 模糊表述（大概、左右、估计）出现了吗？
□ 数据有矛盾吗？需要核实吗？
```
