## Agent职责

### 1. 情报员 (Intel Agent)
- **角色**: 新闻雷达，盯财经热点
- **输入**: 东方财富、新浪财经等新闻源
- **输出**: 重要财经新闻摘要、利好/利空板块
- **触发**: 每日早报，盘中突发事件

### 2. 研读师 (Researcher Agent)
- **角色**: 研报助手，读券商研报
- **输入**: 券商研报、机构评级
- **输出**: 目标价、评级、核心逻辑
- **触发**: 每日研报摘要

### 3. 精算师 (Analyst Agent)
- **角色**: 数据分析师，接入实时行情
- **输入**: mootdx (通达信) + akshare
- **输出**: 行情数据、技术分析、基本面数据
- **触发**: 每日行情、持仓监控

### 4. 守望者 (Watcher Agent) - 投资大师
- **角色**: 投资总监，40年实战经验
- **输入**: 情报员 + 研读师 + 精算师 的输出
- **输出**: 大师级市场分析、可操作的投资建议
- **核心理念**: 利弗莫尔择时 + 巴菲特价值

## 数据源策略

### mootdx (通达信) - 主数据源
- **优势**: 稳定、低延迟、数据完整
- **用途**: 指数K线、个股行情、分时数据
- **配置**: 固定服务器 124.70.199.56:7709

### akshare - 备用/补充数据源
- **优势**: 数据类型丰富（涨跌停、北向资金）
- **劣势**: 网络不稳定，有超时风险
- **用途**: 涨跌停池、北向资金、龙虎榜

## 投资大师体系

### 择时体系 (利弗莫尔)
1. 识别市场主要趋势
2. 等待关键点确认（突破/回踩）
3. 试仓-加仓-止损 标准化
4. 永不逆势加仓

### 选股体系 (巴菲特)
1. 行业龙头，定价权
2. 业绩稳定，现金流优秀
3. 合理估值
4. 管理团队可信

### 仓位管理
- 单只仓位 ≤ 20%
- 强势股可超配，必须设止损
- 不确定性高时，仓位降至30%以下

## 技术栈

- **Python**: 3.8+
- **数据获取**: mootdx, akshare
- **Agent框架**: 自定义Hermes Agent
- **数据存储**: JSON/Markdown
